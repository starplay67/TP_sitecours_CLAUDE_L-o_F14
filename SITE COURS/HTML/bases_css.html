Bases en CSS





Le CSS (Cascaded Style Sheet) sert à donner un style graphique à un html, ils seront donc toujours jumelés. Pour appeller un fichier CSS en HTML, il suffit d'ajouter < link rel="stylesheet" type="text/css" href="nom_css.css"/> dans l'en-tête.

Pour donner un style, on appelle la balise par son nom, puis on ouvre des {} et on lui attribue les styles voulus.
