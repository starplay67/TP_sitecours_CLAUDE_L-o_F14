Les bases

HTML = Hyper Text Mark-up Language

Ainsi, le HTML permet de faire du WYSIWYG (What You See Is What You Get) : créer des sites sans langage particulier, Mozilla Composer, Microsoft Expression Web, etc ...



Les Balises :

Il y a deux types de balise, les balises paires et les balises orphelines.

Les balises paires s'ouvrent, contiennent du texte, et se ferme plus loin ; Une balise ouvrante : < balise > ; une balise fermante : < / balise >

Les balises impaires sont des balises qui servent le plus souvent à insérer un élément à un endroit précis (par exemple une image). Il n'est pas nécessaire de délimiter le début et la fin de l'image. Une balise orpheline s'écrit < balise /> (le slash sert à les différencier des balises paires).

Les Attributs :

Ils viennent les compléter pour donner des informations supplémentaires. L'attribut se place après le nom de la balise ouvrante et a le plus souvent une valeur ; Exemple : < image nom=photo.jpg />, nom de l'image à afficher. Dans le cas d'une balise paire, les attributs sont toujours dans les balises ouvrantes.

Le doctype, la balise html et l'en-tête :

Le doctype est indispensable avant tout car il indique qu'il s'agit d'une page HTML en HTML5.

La balise html (paire) est la première et la dernière balise. Tout ce qui est avant ou après ne sera pas affiché.

L'en-tête head (paire) contiendra obligatoirement la balise meta (impaire).

Commentaires

Pour mettre un commentaire dans un page html, il suffit d'inscrire < !-- commentaire !-->; ils ne seront pas affichés dans le rendu.
