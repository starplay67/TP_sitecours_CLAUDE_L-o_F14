Les liens





Qu'est-ce qu'un lien et comment les utiliser ?

Un lien est un texte sur lequel on peut cliquer pour se rendre sur une autre page. On peut faire un lien d'une page a.html vers une page b.html, mais on peut aussi faire un lieu vers un autre site.

Balise < a > (paire); attribut href ; valeur de l'attribut est URL de la page ciblée.

Exemple : < a href="https://bungie.net">Bungie< /a>

Ce sont des liens absolus car on indique l'adresse complète. Ils sont généralement à éviter.

Il y a les liens relatifs (par rapport à l'endroit où on est) qui permet de prendre directement dans les fichiers. Avec ".." on remonte dans l'arborescence, et avec le "/" on redescend dans l'arborescence.

Un lien vers une ancre

C'est un lien vers un endroit dans la même page. On utilise une balise commune comme attribut un id dont la valeur sera le nom de l'ancre, puis on utilise un lien normal, sauf que la valeur du href sera # suivi du nom de l'ancre.
